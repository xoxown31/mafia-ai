# 🚀 Mafia AI 재학습 시작 가이드

## ✅ 적용된 최적화 요약

### 1. 하이퍼파라미터 조정
- **Learning Rate**: 0.0001 → 0.00005 (50% 감소)
- **Entropy Coefficient**: 0.05 → 0.01 (80% 감소)

### 2. 보상 체계 재설계
- **승리 보상**: +30 → +100 (333% 증가)
- **패배 페널티**: -15 → -50 (333% 증가)
- **중간 보상**: 승리의 10% 이내로 대폭 하향 조정

### 3. 관측 공간 확장
- **기존**: 152차원
- **새로운**: 168차원 (누적 피처 16차원 추가)
  - 누적 지목 횟수 (8)
  - 누적 투표 횟수 (8)

### 4. 모델 아키텍처 옵션
- **기본 MLP**: 3-layer fully connected network
- **GRU 확장**: 시계열 추론을 위한 GRU 레이어 추가 (선택 사항)

---

## 📋 실행 방법

### 방법 1: 기본 MLP로 재학습 (권장)

```bash
python main.py --mode train --agent ppo --episodes 5000
```

**예상 소요 시간**: 약 30-60분 (CPU 기준)

### 방법 2: GRU 모델로 학습 (고급)

```bash
python main.py --mode train --agent ppo --episodes 5000 --use-gru
```

**예상 소요 시간**: 약 45-90분 (CPU 기준)

### 방법 3: 더 많은 에피소드 (권장: 10,000+)

```bash
python main.py --mode train --agent ppo --episodes 10000
```

## 💡 학습 팁

### 1. 학습이 느리게 느껴진다면
- `--episodes` 값을 5,000 이상으로 증가
- GPU가 있다면 PyTorch CUDA 활성화 고려

### 2. 승률이 50%에 도달하지 못한다면
- 10,000 에피소드까지 학습 연장
- `ENTROPY_COEF`를 0.005로 추가 감소 (`config.py` 수정)

### 3. 보상 그래프가 불안정하다면
- 정상적인 현상 (마피아 게임의 확률적 특성)
- 이동평균 추세가 우상향하면 OK

## 🐛 문제 해결

### 문제 1: "ModuleNotFoundError"
```bash
pip install -r requirements.txt
```

### 문제 2: "CUDA out of memory"
CPU 모드로 실행되도록 PyTorch 재설치:
```bash
pip install torch --index-url https://download.pytorch.org/whl/cpu
```

### 문제 3: "All actions masked out!"
정상 작동 중입니다. 드물게 발생하는 엣지 케이스이며 자동 복구됩니다.
